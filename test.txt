Тест 1. Тестування GET з пустим кешом. Очікуємо 404 Not Found.
curl localhost:9999/200
 
Тест 2. Тестування PUT, створення файлу в кеші із локального файлу
curl -X PUT --data-binary @200.jpeg localhost:9999/200

Тест 3. Тестування GET з файлом з кешу, створений за допомогою PUT та локального файлу. Очікуваний результат 200 OK як вихідне зображення результату
curl localhost:9999/200 --output Getting200FromCache.jpeg
 
Тест 4. Тестування DELETE, видалення файлу з очікуваним кодом 200 ОК та повторне використання для видалення того ж самого файлу для отримання статус коду 404 NOT FOUND
curl -X DELETE http://localhost:9999/200
 
Тест 5. Тестування Обробки інших запитів, очікуваний результат 405 код Method Not Allowed
curl -v -X POST http://localhost:9999/200
